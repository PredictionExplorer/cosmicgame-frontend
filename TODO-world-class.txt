WHAT "WORLD-CLASS" MEANS
================================================================================

A world-class repository would have:
- Zero magic numbers (all configurable via constants)
- No file over 300 lines
- All data fetching through React Query (no manual useEffect fetching)
- Shared API error handling (DRY)
- Fully typed (zero `as unknown as` in production code)
- 80%+ test coverage with meaningful tests
- Interaction, error state, and accessibility tests
- All external links secured with rel="noopener noreferrer"
- Every exported function documented with JSDoc
- CI that passes on every commit
